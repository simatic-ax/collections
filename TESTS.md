# Creating a tests project

## Preconditions

1. Create a library project
2. Open a shell in the repo directory


## Steps for creation

1. `apax init --app @ax-showcase/my-lib.tests`
2. `cd ax-showcase-my-lib.tests`
3. `touch TESTS.md`
4. `nano .gitignore`
5. `nano apax.yml`
6. `cd ..`


## Steps for test execution

`apax run test`
